# C言語 練習課題

````admonish info "C言語の実行方法等"
C言語のソースコード作成後、実行するために**コンパイル**と**リンク**という処理を行い、**実行ファイル**を作成する必要があります。  
コンパイラ・リンカに`GCC`を使用します。  

ソースコードが置かれているディレクトリに移動します。ここでは例として`~/program/`というディレクトリの中にある`helloworld.c`というファイルを実行したいとします。

```c : helloworld.c
#include <stdio.h>

int main(void){
    printf("Hello,World!\n");

    return 0;
}
```

```shell : ubuntu22.04 (WSL)
$ cd ~/program/

$ ls
helloworld.c

$ gcc helloworld.c -o helloworld

$ ls
helloworld  helloworld.c

$ ./helloworld
Hello,World!

```

`gcc ソースコードファイル名 -o 実行ファイル名`でコンパイルとリンクを行います。  
実行するときは`./実行ファイル名`で実行します。  

`math.h`を使用する場合は、コンパイル時に`-lm`をつける必要があります。  
`gcc ソースコードファイル名 -lm -o 実行ファイル名`でコンパイルとリンクを行います。
````

## C言語(基礎編)

### 円の面積と円周

実数の半径 \\(r\\) である円の面積と円周を求めるプログラムを作成してください。ただし、 \\(r\leqq0\\) である場合は、プログラムは**終了せずに**再入力を促してください。  
また、**入力部・処理部・出力部**の3つで分かれるようにプログラムを作成してください。  

```shell : 実行例
$ gcc ci_ar_r.c -lm -o ci_ar_r

# r = 3.0 の場合
$ ./ci_ar_r
Please enter r (r > 0): 0
Please enter r (r > 0): -1.5
Please enter r (r > 0): 3
l : 18.849556
s : 28.274334

# r = 1.5 の場合
$ ./ci_ar_r
Please enter r (r > 0): 1.5
l : 9.424778
s : 7.068583
```

````admonish tip "定義された円周率を使用する"
円周率は各自で`3.14`を使用しても良いですが、`math.h`に`M_PI`という名前で小数点以下20桁ほどまで定義されています。  

```c : 半径rの円の面積と円周を表示するプログラム
#include <stdio.h>
#include <math.h>
#include <stdlib.h>

int main(void){
    double r, l, s;

    // 入力部
    scanf(" %lf", &r);

    // 処理部
    l = 2 * M_PI * r;
    s = M_PI * r * r;

    // 出力部
    printf("l : %f\n", l);
    printf("s : %f\n", s);

    return EXIT_SUCCESS;
}
```

```shell : 実行例
$ gcc ci_ar.c -lm -o ci_ar

# r = 3.0 の場合
$ ./ci_ar
3
l : 18.849556
s : 28.274334

# r = 1.5 の場合
$ ./ci_ar
1.5
l : 9.424778
s : 7.068583
```
````

### 2次方程式の解

実数の係数 \\(b,c\\) を持つ2次方程式 \\(x^2+bx+c=0\\) の実数解は、判別式 \\(D=b^2-4c\\) の値によって以下の表のようになります。  

| D | 解の個数 | 解 |
| :-: | :-: | :-: |
| \\(D>0\\) | \\(2\\) | \\(\frac{-b\pm\sqrt{D}}{2}\\) |
| \\(D=0\\) | \\(1\\) | \\(\frac{-b}{2}\\) |
| \\(D<0\\) | \\(0\\) | 実数解なし |

実数の係数 \\(b,c\\) を入力すると、2次方程式の解を求め、出力するプログラムを作成してください。実数解が存在しない場合は `no real roots` と出力してください。  
また、**入力部・処理部・出力部**の3つで分かれるようにプログラムを作成してください。

```shell : 実行例
# b = 3.5, c = 1.2 の場合
$ ./qu_eq
3.5
1.2
x = -0.385266 , -3.114734

# b = -2.0, c = 1.0 の場合
$ ./qu_eq
-2
1
x = 1.000000

# b = 2.0, c = 4.0 の場合
$ ./qu_eq
2
4
no real roots
```

````admonish tip "平方根の計算"
平方根をC言語で計算するためには`math.h`にある`sqrt()`を使用します。  

以下は \\(\sqrt{2}\\) を表示するプログラムです。

```c : sqrt_2.c
#include <stdio.h>
#include <math.h>

int main(void){
    double sqrt_2 = sqrt(2);
    printf("%f\n", sqrt_2);

    return 0;
}
```

```shell : 実行例
$ gcc sqrt_2.c -lm -o sqrt_2

$ ./sqrt_2
1.414214
```
````

### 放物線とx軸との交点

整数の係数 \\(a,b,c\\) を持つ放物線 \\(y=ax^2+bx+c\\) とx軸との交点を求め、小数点以下1桁で出力するプログラムを作成してください。  
また、**入力部・処理部・出力部**の3つで分かれるようにプログラムを作成してください。  
ヒント：「新基礎数学 改訂版 p.82」

```shell : 実行例
# a = 1 b = -2, c = -3 の場合
$ ./pa_in
1
-2
-3
(3.0, 0), (-1.0, 0)

# a = 2 b = -8, c = 9 の場合
$ ./pa_in
2
-8
9
no intersections

# a = -1 b = 4, c = -4 の場合
$ ./pa_in
-1
4
-4
(2.0, 0)
```
